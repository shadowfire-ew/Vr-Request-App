This folder contains the Android Studio projects for both apps

Admin:
	this is the admin app, which manages open requests
	the admin will see a list of open requests and can select one for more info and actions
	in this more actions page, the admin can deny or approve the request

Client:
	this is the client app, which can send requests
	the client can also change their name or cancel the request
	the client should recieve a notification once their request is apporved or denied (WIP)
